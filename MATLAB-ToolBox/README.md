# Muse 数据流可视化与采集系统

基于 MATLAB 的实时生物信号数据流可视化与采集工具，支持 **LSL（Lab Streaming Layer）** 和 **OSC（Open Sound Control）** 两种协议。

## 文件说明

| 文件 | 功能 |
|------|------|
| `main.m` | 主界面入口，提供 LSL / OSC 启动按钮和运行日志面板 |
| `lsl.m` | LSL 协议多流实时查看器与记录器 |
| `osc.m` | OSC/UDP 协议数据流自动发现与实时绘图 |

## 快速开始

1. 打开 MATLAB，将工作目录切换到 `E:\MyApp\`
2. 在命令窗口输入 `main` 启动主界面
3. 点击 **启动 LSL** 或 **启动 OSC** 进入对应的数据流模式

也可以直接运行子模块：

```matlab
lsl    % 启动 LSL 模式
osc    % 启动 OSC 模式（运行时需输入 UDP 端口和 IP）
```

## 功能特性

- **自动发现数据流**：自动扫描并识别 LSL 流或 OSC/UDP 数据包，无需手动配置
- **多通道实时波形**：每条流的所有通道以独立子图显示，使用 `animatedline` 实现低延迟绘制
- **智能 Y 轴缩放**：匹配到预设信号时使用固定范围，否则根据数据自适应调整
- **数据录制**：支持将当前显示的流数据保存为 CSV 文件，包含时间戳和各通道数值
- **历史回溯**：切换流时自动恢复缓冲区内的历史波形，避免数据丢失
- **27 种预设信号**：覆盖脑电（EEG）、陀螺仪（Gyro）、加速度计（Accel）、肌电（JawClench）、眨眼（Blink）、脉搏波（PPG）、温度（Therm）、电池（Batt）以及 Alpha/Beta/Delta/Theta/Gamma 频段能量等

## 依赖要求

- MATLAB R2020b 或更高版本
- **LSL 模式**：需安装 [liblsl-Matlab](https://github.com/sccn/liblsl-matlab) 并将其添加到 MATLAB 路径
- **OSC 模式**：使用 MATLAB 内置的 `udpport`，无需额外依赖

## 使用说明

### LSL 模式

1. 点击 **启动 LSL**，程序自动扫描局域网内所有 LSL 流（超时 5 秒）
2. 程序会根据流的 `type` 字段自动匹配预设信号；若有未匹配的流，会弹出映射界面供手动选择
3. 通过左上角下拉菜单切换不同流，查看对应波形
4. 点击 **开始记录** → 选择保存目录 → 输入文件名，即可开始录制 CSV 文件
5. 点击 **停止记录** 结束录制，文件句柄自动关闭

### OSC 模式

1. 点击 **启动 OSC**，弹出对话框输入：
   - **UDP 端口号**：1-65535 之间的整数，默认 7000
   - **本地 IP 地址**：留空表示监听所有网络接口
2. 程序自动解析 OSC 数据包，根据地址路径（如 `/JawClench`）的最后一段识别信号名称并匹配预设范围
3. 新出现的 OSC 地址会自动添加到下拉菜单，历史流保留不丢失
4. 数据录制操作同 LSL 模式

### OSC 数据包格式

- 符合标准 OSC 协议
- 支持 `f`（float32）和 `i`（int32）两种数值类型
- 若第一个参数为字符串类型的时间戳，会自动跳过
- 示例地址格式：`/Muse/JawClench` 或 `/EEG`

### 录制文件格式

```csv
Timestamp,Ch1,Ch2,Ch3,...
1620000000.123456,0.123,-0.456,0.789,...
1620000000.133456,0.124,-0.455,0.790,...
```

- 第一行为表头：`Timestamp` + 各通道名
- 每行一个采样点：时间戳（秒） + 对应通道的数值
- 数值保留 6 位小数

## 可调参数

### lsl.m

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `window_time` | 2 | 波形显示窗口时长（秒） |
| `data_gain` | 1 | 数据显示增益，作用于去均值后的数据 |
| `buffer_size` | 20000 | 每条 animatedline 的最大点数 |
| `max_mean_points` | 5000 | 用于计算均值的最近采样点数 |
| `print_interval` | 100 | 每隔多少个样本打印一次调试信息 |
| `MANUAL_TIME_SCALE` | 1e-6 | 时间戳转换因子（微秒 → 秒） |

### osc.m

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `window_time` | 2 | 波形显示窗口时长（秒） |
| `DEBUG_MODE` | true | 是否在命令窗口输出调试信息 |
| `MAX_POINTS_PER_LINE` | 5000 | 每条 animatedline 的最大点数 |

## 注意事项

- LSL 模式下切换数据流前需先停止录制，否则会弹出提示
- OSC 模式的 UDP 端口如被占用，程序会报错并提示检查
- 关闭图形窗口即退出程序；若正在录制中，退出前会自动关闭文件句柄，不会损坏已保存的数据
- 同一 OSC 地址的通道数发生变化时，缓冲区会自动清空重建
- 若发现 LSL 流的时间戳与实际不符，可调整 `MANUAL_TIME_SCALE` 参数
